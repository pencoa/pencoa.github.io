在服务器上安装Neo4j

连接服务器时没有图形界面与浏览器，需要在shell中下载与安装Neo4j.
在shell中使用wget下载

NEWVERSION是你需要下载的Neo4j版本号，版本号的列表可以从这个网址中得到。

依照官网的指导来安装Neo4j

  1. 下载安装包，在上一步当中已经完成。

  2. 使用命令解压安装包，解压的顶层目录称为。 NEO4J_HOME

  3. 进入NEO4J_HOME目录，运行命令启动数据库。

  4. 通过Ctrl-C来停止数据库运行。

在运行Neo4j的过程中，如果遇到这种bug

这是因为Neo4j依赖java内核运行
可以先依照Oracle教程安装java
然后设置JAVA_HOME的路径。

  1. 寻找openjdk的路径

  2. 修改配置，可能需要sudo权限

  3. 通过 i 进入写入模式

  4. 增加设置到文件中

  5. 重新打开shell或者使用配置
